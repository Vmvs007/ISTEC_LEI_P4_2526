## Bem-vindo(a)! 👋  

Este repositório reúne as resoluções das fichas práticas desenvolvidas no âmbito da UC de **Programação IV (PROG4)**, integrada na **Licenciatura em Engenharia Informática** do **ISTEC Porto**.

A unidade curricular aprofunda o desenvolvimento de software em **Python**, com foco em:

- Programação Orientada a Objetos (POO)
- Design Patterns
- Programação concorrente
- Sistemas distribuídos
- Docker
- Desenvolvimento colaborativo e testes

O objetivo deste repositório é facilitar o estudo, consolidar o conhecimento prático e promover boas práticas de programação em Python, preparando os estudantes para desafios reais de engenharia de software.

---

## 📂 Estrutura do Repositório  

As resoluções estão organizadas por ficha prática (PL), acompanhando a sequência dos conteúdos lecionados ao longo do semestre:

Cada pasta poderá conter:
- Código-fonte em Python
- Exercícios resolvidos
- Exemplos práticos
- Eventuais ficheiros de apoio

---

## 🔍 Temas Abordados  

Ao longo das fichas práticas são trabalhados os seguintes tópicos:

- Fundamentos avançados de Python  
- Programação Orientada a Objetos  
- Design Patterns (Criacionais, Estruturais e Comportamentais)  
- Testes unitários e Test Driven Development (TDD)  
- Threads e concorrência  
- TCP, UDP e Web Sockets  
- Sistemas Distribuídos  
- Docker e Docker Compose  
- RESTful APIs
